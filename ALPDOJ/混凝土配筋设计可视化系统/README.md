# 混凝土配筋设计可视化系统
基于 Qt6 开发的单轴双轮系统下混凝土简支梁自动配筋计算工具，支持参数输入、规范配筋计算、多视图绘图、HTML设计报告生成。
# 环境依赖
- 开发工具：Visual Studio 2022
- Qt 版本：Qt 6.11.0
# 编译与运行
1. 打开项目：使用 Visual Studio 打开项目根目录下的 `.sln` 解决方案文件 
2. 环境校验：确保VS已正确配置Qt6开发环境，Qt版本与项目平台（x64）匹配 
3. 编译项目：在VS顶部工具栏，选择「Release」模式 + 「x64」平台，点击「生成」→「重新生成解决方案」
4. 启动程序：编译无报错后，点击VS「本地Windows调试器」，即可启动程序
# 使用说明
1. 输入设计参数：在程序左侧输入框，依次填写梁跨度（单位：mm）、荷载值（单位：kN）、车轮间距（单位：m） 
2. 保存参数：点击**传递参数**按钮，程序会校验并保存输入参数 
3. 计算与绘图：点击**显示结果**按钮，程序自动执行配筋计算，并在右侧窗口绘制梁体配筋图纸 
4. 生成设计报告：点击**report**按钮，选择保存路径，即可生成HTML格式设计报告，生成后自动打开
==注意先输入参数再点击传递参数按钮==

# 项目结构
	├── README.md                   //帮助文档

	├── main.cpp                    //程序入口，Qt 应用初始化

	├── mainwindow.h/mainwindow.cpp //主窗口：界面布局、按钮交互、报告生成逻辑

	├── stdSize.h/stdSize.cpp       //标准化桥梁宽度和高度尺寸

	├── conParams.h                 //全局数据：桥梁设计相关常量参数

	├── designEle.h                 //存储用户输入的数据和程序输出的数据

	├── rebar.h/rebar.cpp          //钢筋类，计算钢筋相关参数

	├── Individual.h              //遗传算法中的个体类，存储个体数据

	└── 混凝土配筋设计.vcxproj # VS 项目工程配置文件
